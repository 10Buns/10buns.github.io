代账计费系统

作为公司的核心系统之一，主要提供客户代账计费、数据汇总、报表展示、运营财务日常使用等。
我在此项目中负责计费系统的技术改造、新业务支撑开发。

原代账系统使用 vert.x 、jpa 等框架开发，但由于前期需求或规划问题，与公司其他项目存在技术框架不匹配，性能不可靠等问题。

接手项目后，首先拆分两个方向保证系统的持续运行和迭代, 1) 新的业务需求使用公司已全面推广的 spring cloud 框架开发 2）原项目上优化现有业务功能的性能，优化完成后迁移至新的系统。

经过两个月的快速迭代，整个计费系统性能有了非常大的提升。相关API接口从3~15s降为500ms内，报表服务也做到了秒级别, 给运营支撑带来了更快速的响应。
