spring-data-redis序列化问题

Jun 16, 2017

昨天同事通过 spring-data-redis 操作 redis，取值的时候发现一个奇怪的现象，取出的key前面会有一串
\xac\xed\x00\x05t\x00\tb 这种玩意

0x01 原因分析

第一眼就感觉是序列化、反序列化问题。那么直接查看 RedisTemplate 源码

可以看到 spring 默认是采用 JdkSerializationRedisSerializer 对 key 和 value 进行序列化处理，其默认支持的是对象序列化，但 spring-data-redis 支持的是:StringRedisSerializer, JacksonJsonRedisSerializer, OxmSerializer
而后两者都是基于 StringRedisSerializer 的高级封装。

0x02 解决问题

既然问题找到了，那么久手动设置序列化方式为 StringRedisSerializer
